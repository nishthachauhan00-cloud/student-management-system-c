# student-management-system-c
console-based student management system in c using file handling

## Description
A console-based Student Management System written in C.
The project uses structures and binary file handling to store and manage student records.

## Features
- Add student record
- Display all students
- Search student by roll number
- Update student details
- Delete student record

## Technologies Used
- C Programming
- File Handling
- Structures
- Functions

## How to Run
1. Compile using a C compiler (GCC)
2. Run the executable file
